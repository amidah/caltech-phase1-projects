Template driven refers to an approach where most of the form logic and structure are defined within the template(HTML), rather than the component class.

Here we are using Angular's built-in directives like ngModel and ngForm directly in the HTML to manage form data and submission.

ngModel: It automatically binds the input fields to a model (in the component). This enables two-way data binding, so the form fields reflect changes in the model and vice versa.

ngForm: This directive automatically creates a form group to track the entire form. Angular uses it to monitor the form's state

In Angular, pipes are a powerful feature used to transform data in templates. They allow you to apply various transformations like formatting, filtering, or modifying data before it's displayed to the user. Pipes are very useful for tasks like formatting dates, converting text to uppercase/lowercase, transforming numbers into currency formats, and much more.


{{ hello }}

{{ hello | uppercase }}

{{ hello | lowercase }}


{{ today }}

{{ today | date }}

{{ today | date:'fullDate' }}


{{ currencyNum }}

{{ currencyNum | currency: 'USD' }}
