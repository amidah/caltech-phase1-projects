HTML

HTML is the standard markup language of all the docs that are designed to be displayed in a web browser. It can be assisted by some technologies such as Cascading Style Sheets (CSS) and some scripting languages such as Javascript.

HTML History

Sir Tim Berners-Lee developed HTML in late 1991 but did not formally release it. HTML 2.0 was released in 1995. HTML 4.01, a significant version, was released in late 1999.
